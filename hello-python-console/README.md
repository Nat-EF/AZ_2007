# Hello Python Console Application

This is a simple Python console application that prints "Hello World" to the console.

## Project Structure

```
hello-python-console
├── src
│   └── main.py
├── requirements.txt
├── .gitignore
└── README.md
```

## How to Run

1. Make sure you have Python installed on your machine.
2. Navigate to the project directory.
3. Run the application using the following command:

   ```
   python src/main.py
   ```

This will execute the script and print "Hello World" to the console.
